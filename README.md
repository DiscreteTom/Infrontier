# Infrontier

> Powered by [electron-nuxt](https://github.com/michalzaq12/electron-nuxt).

A simple AWS S3 client written with Electron.

## Build Setup

```bash
# install dependencies
yarn install

# serve app with hot reload
yarn dev

# build electron application for production
yarn build
```
